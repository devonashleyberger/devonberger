Project

Belkin

Linksys Shield Phase 2: Network Security

Background

Linksys Shield is a premium subscription software service that blocks unwanted content and provides an additional layer of digital protection for connected devices on a Linksys network. This assignment was to 1. decouple the malicious websites threat blocker from the parental controls feature set and 2. design a new service specifically for network level security.

Some exploration went into this project prior to my contract with Belkin. So—to get myself acquainted with the work and quickly understand some assumptions, I refined some of the more promising explorations and began testing.

Implementation Details

The discovery phase revealed the user’s need for evidence of success, specifically activity logs for threats that have been blocked by the service. I worked with the engineering team to design logic that would get the user the most current and relevant logs while keeping the “expense” down for the engineering team.

Once the existing behavior of the system was understood, the cloud costs could be evaluated and accurately communicated to the rest of the UX team. From there, we confidently returned a considered and vetted design to the engineering team for implementation.
